This repository contains the code I used for numerical simulations of the CGLE. Below is a description of each file:


ginzburg-landau modA : Generates plots of |A| vs. x every 50.0 time units, saves them as .png files in the same folder as the program

ginzburg-landau x_t plots: Generates a colour-mapped x-t plot where colours indicate relative size of Re(A)

ginzburg landau-phase_plots: Generates a series of phase plots for the complex phase of A(x,t). This code and its plots were not used in the final report

ginzburg-landau_varying_c_g: Generates a series of plots every 1 time unit of Re(A) vs. x. Also saves one of these plots every 50 units
